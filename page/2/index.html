NLPEexp3:Text Match via Attention

2020-05-12

Enhanced LSTM for Natural Language Inference

  1. 使用BiLSTM预处理，当前位置的两个反向hidden state拼接得到当前位置的最终hidden state。
  2. sequential inference model 收集的是词及其上下文信息，tree LSTM收集的是短语和从句间的信息。
  3. 使用soft attention，计算两个premise和hypothesis间的attention权重。
  4. 内积计算相似度。

  6. 最后使用3层的MLP（tanh做hidden layer的激活函数）。

  7. 整个模型取交叉熵损失函数。

2020-02-10
Hash Learning

针对跨模态检索、基于哈希的字典学习

Xu, Xin-Shun. “Dictionary learning based hashing for cross-modal retrieval.” Proceedings of the 24th ACM international conference on Multimedia. ACM, 2016.

Integrated Hash

  1. 之前：把不同模态的数据直接投影到同一个空间，但是由于某些语义上相近的数据在隐空间(latent space)投影后难以靠近造成误差。
  2. 改进：并不是直接对不同模态的数据进行投影，而是首先先为每个模态学习出字典并构造出数据的稀疏表示，再对稀疏表示进行线性投影。
